自适应学习助手

基于 LangGraph 的智能自适应学习系统 | Adaptive Learning Agent

Github仓库: https://github.com/codewithwu/adaptive-learning-agent
📌
项目定位
  • AI 驱动的自适应学习辅导系统
  • 核心理念：费曼学习法 + 间隔检测效应
  • 通过出题-验证-教学的闭环循环帮助掌握知识
  • LangGraph 状态机编排多步骤 AI 工作流
✨
核心亮点
  • 自适应路径：3 个递进检查点（基础 → 应用 → 掌握）
  • 语义分块：向量检索实现笔记精准出题
  • 智能补充：Tavily 搜索补充学习资料
  • 量化评估：0-100% 评分，低于 70% 自动教学
🛠️
技术栈
  • Agent 框架：LangGraph 1.1.10+ 状态机
  • LLM 编排：LangChain 1.2.17+
  • 后端服务：FastAPI + Uvicorn
  • 向量嵌入：Ollama Embeddings 本地
  • 网络搜索：Tavily Search API

📊
LearningState 状态定义
  • topic：学习主题
  • goals：学习目标列表
  • context：原始笔记内容
  • context_chunks：Annotated[list, operator.add] 累加语义
📦
9 个 Pydantic 模型
  • LearningCheckpoint：单个学习检查点
  • LearningVerification：答案验证结果
  • FeynmanTeaching：费曼教学内容
  • QuestionOutput：生成的题目
🔧
10 个图节点
  • generate_checkpoints：生成递进检查点
  • chunk_context：OllamaEncoder + StatisticalChunker
  • context_validation：余弦相似度 + Top-K=3
  • verify_answer：基础模型失败自动降级
🔀
条件路由 (routes.py)
  • route_context：chunk_context | generate_query
  • route_search：search_web | generate_question
  • route_verification：next_checkpoint | teach_concept | END
  • route_teaching：next_checkpoint | END
🧠
向量检索与上下文
  • ContextStore：基于 InMemoryStore 封装
  • 语义分块：128-512 tokens 粒度
  • 检索策略：余弦相似度 + Top-K=3
  • 编码器：OllamaEncoder 本地嵌入
🏭
双模型 LLM 工厂
  • basic_model：LongCat（美团）日常任务
  • advanced_model：百灵（Ling）降级方案
  • 8 个提供商：ollama/nvidia/zhipu/bailing...
  • 单例模式：@lru_cache 缓存实例
🛡️
自定义中间件
  • dynamic_model_selection：消息 >10 条切换高级模型
  • comprehensive_tool_filter：多维度过滤工具
  • handle_tool_errors：错误捕获与友好提示
  • user_role_prompt：角色动态调整提示词
📡
API 端点
  • POST /api/sessions：创建学习会话
  • POST /resume：恢复中断会话
  • GET /state：获取状态快照
  • Command(resume=...)：中断恢复机制
📝
提示词工程（6 个模板）
  • 检查点生成：3 个递进检查点
  • 上下文验证：严格 YES/NO 输出
  • 答案验证：多维度评估
  • 费曼教学：无术语简化 + 类比
