LangGraph 多Agent系统 ADHD友好 学习助手 ReAct推理

学习智能体

基于 LangGraph + LangChain 的多 Agent 智能学习规划系统

Github仓库: https://github.com/codewithwu/academic-task-learning-agent
📌
项目定位
  • 为学生提供个性化学习规划、资料生成和学习指导
  • 特别关注 ADHD 学生和需要兼顾体育等事务的学习者
  • 通过 AI 协调多个专业 Agent 实现一站式学习辅助
💡
核心价值
  • Planner Agent：分析日历和任务，生成 ADHD 友好计划
  • NoteWriter Agent：根据学习风格生成结构化笔记
  • Advisor Agent：针对学业情况提供定制建议
  • Coordinator Agent：统一协调调度多 Agent 并行工作
⚙️
技术栈
LangGraph LangChain FastAPI Ollama OpenAI NVIDIA
  • Agent 编排：langgraph >= 1.1.10
  • LLM 抽象：langchain >= 1.2.17
  • 速率限制：langmem >= 0.0.30
  • Web 框架：fastapi >= 0.136.1
🌐
多后端 LLM 支持
  • ollama：本地部署，零成本
  • nvidia：NVIDIA NGC 云端算力
  • openai：GPT-4/GPT-3.5
  • deepseek、zhipu、bailing、siliconflow 等

🔧
核心模块 — AcademicState
  • 继承 MessagesState，统一状态容器
  • 使用 dict_reducer 合并嵌套字典
  • 支持 profile/calendar/tasks/results 字段
  • 实现字典递归合并，避免状态覆盖
🧠
核心模块 — ReActAgent
  • Thought：分析当前状态和上下文
  • Action：决定下一步工具/方法
  • Observation：记录动作结果
  • Plan：制定后续计划
📅
Planner Agent 三阶段
  • calendar_analyzer：分析可用时间块、冲突检测、精力模式
  • task_analyzer：优先级、时间估算、依赖关系
  • plan_generator：生成综合学习计划
  • 内置 Few-shot 示例，ADHD 友好设计
📝
NoteWriter Agent 两阶段
  • analyze_learning_style：分析学习风格偏好
  • generate_notes：生成 Markdown 个性化笔记
  • 输出包含记忆强化技巧嵌入
  • 适配学生学习风格偏好
🎓
Advisor Agent 两阶段
  • analyze_situation：分析当前学业情况
  • generate_guidance：生成学习指导建议
  • 指导内容：立即可采取的行动步骤
  • 日程优化、精力管理、应急计划
⚡
AgentExecutor 并发引擎
  • 使用 asyncio.gather 实现真并发
  • 多 Agent 并行执行
  • 使用 return_exceptions=True 捕获异常
  • 显著提升系统响应速度
🏭
LLMFactory 多后端工厂
  • 支持 9+ 种 LLM 提供商
  • 工厂模式 + @lru_cache 单例
  • InMemoryRateLimiter(0.1) 全局限流
  • 防止 API 过载
数据模型
学生档案 profile.json
personal_info / academic_info / learning_preferences / productivity_tools
日历事件 calendar_events.json
events: id / summary / start / end / metadata
任务 task.json
tasks: id / title / due / status / priority / adhd_support
