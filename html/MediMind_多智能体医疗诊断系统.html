PDFProcessor 服务

PDF 文件处理与文本提取

  • save_pdf() 保存并返回 UUID
  • load_pdf() 提取文本内容
  • get_file_path() 获取文件路径
  • 使用 PyPDFLoader
  • 文件存储于 uploads/
⚙️
TaskManager 服务

管理分析任务生命周期

  • AnalysisTask 任务数据模型
  • create_task() 创建新任务
  • get_status() 查询进度
  • get_result() 获取结果
  • set_failed() 错误处理

API Routes 接口

FastAPI 异步接口定义

  • POST /api/upload 上传 PDF
  • GET /api/task/{id} 查询状态
  • GET /api/result/{id} 获取结果
  • GET /api/health 健康检查
  • Pydantic 数据校验

关键设计亮点

🔀 并行 + 串行混合

三个专科 Agent 使用 ThreadPoolExecutor 并行执行，最后串行汇总至多学科团队

⚡ 异步非阻塞

FastAPI 异步框架 + 后台线程池，上传响应即时返回，分析不阻塞主请求

📦 Pydantic 校验

所有 API 输入输出使用 BaseModel 类型校验，自动生成 OpenAPI 文档
