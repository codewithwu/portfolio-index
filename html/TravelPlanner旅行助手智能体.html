Travel Planner 旅行助手智能体

基于 LangGraph StateGraph 架构的全生命周期 AI 旅行规划助手，覆盖行前规划、行中执行、行后回顾三大阶段

Github仓库: https://github.com/codewithwu/travel_planner
LangGraph StateGraph 多 Agent 双层记忆 Pydantic
01

项目概览

  • 核心定位：端到端旅行智能体系统
  • 三阶段管理：行前规划、行中执行、行后回顾
  • 技术架构：LangGraph StateGraph 节点化流程编排
  • 测试覆盖：42+ 单元测试用例

02

技术栈

  • 语言：Python 3.13 最新稳定版
  • 智能体框架：LangGraph ≥1.2.1 [!!应为1.0.8+]
  • LLM 抽象层：LangChain ≥1.2.17
  • 数据建模：Pydantic ≥2.0
  • 包管理：uv（高性能）
  • 代码质量：Ruff + mypy
  • 测试框架：pytest ≥9.0
03

架构设计

  • 架构模式：扁平图 + 中央路由器
  • 状态模型：TravelState (dict 子类)
  • 执行方式：每节点单次执行后流向 END
  • 多轮驱动：main.py 循环调用 graph.invoke()
START → router_node → route_by_phase_and_step ├── requirement_collection → END ├── plan_generation → END ├── plan_confirmation → END ├── execution_guidance → END ├── dynamic_adjustment → END ├── itinerary_recording → END ├── experience_review → END ├── content_sharing → END └── feedback_collection → END
04

双层记忆系统

  • 短期记忆：InMemorySaver 单次会话持久化
  • 长期记忆：InMemoryStore 跨会话用户画像
  • 隔离策略：namespace=("user_profiles", user_id)
  • 更新策略：写入时合并（merge-on-write）

05

行前阶段

  • 需求采集：渐进式多轮提取 JSON 结构化数据
  • 行程生成：Mock 目的地服务 + LLM 生成计划
  • 计划确认：LLM 分析反馈判断确认/修改
  • 容错机制：_extract_json() 正则提取有效 JSON
collected_info = { "destination": "", "travel_dates": "", "duration_days": 0, "num_people": 0, "total_budget": "", "preferences": [] }
06

行中阶段

  • 执行引导：根据 current_day 获取当日计划
  • 天气整合：实时天气数据（Mock）穿衣建议
  • 动态调整：L1-L2 级别行程变更
  • 行程记录：用户消息作为当日笔记
  • 结束标记：[TRIP_ENDED] 触发阶段转换
07

行后阶段

  • 体验回顾：对比计划 vs 实际行程生成总结
  • 内容分享：朋友圈风格文案（100-200字）
  • 反馈收集：满意度评分、亮点、改进建议
  • 流程终止：current_step = "end"
08

数据层设计

  • Activity：时间段、活动名称、地点、费用
  • DayPlan：日期、主题、活动列表、餐饮
  • TravelPlan：行程、预算明细、行李清单
  • UserProfile：偏好、预算敏感度、饮食限制
class TravelPlan(BaseModel): destination: str dates: str duration: int pace: str # relaxed/moderate/intensive days: list[DayPlan] budget_breakdown: dict packing_list: list[str]

10

可复用智能体模块

  • StructuredOutputAgent：三阶段降级策略
  • 中间件：动态模型选择、工具过滤、错误处理
  • LLMFactory：统一工厂类
  • 日志引擎：Rich 控制台渲染

11

设计决策与技术亮点

  • 扁平图选择：降低复杂度，易调试
  • dict 子类：兼容 LangGraph + 默认值初始化
  • JSON 容错：正则提取有效 JSON
  • 结束标记：[TRIP_ENDED] 轻量级通信
  • 解耦设计：langchain_agent 独立复用
